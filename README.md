# Computer Networks Lab Assignments

This repository contains my Computer Networks course lab assignments completed using Cisco Packet Tracer. These labs demonstrate practical implementation of networking concepts including IP addressing, subnetting, routing protocols, and network troubleshooting.

## 📋 Lab Overview

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab 01](./lab-01-basic-addressing) | Basic IP Addressing | Introduction to IP addressing and basic network topology |
| [Lab 02](./lab-02-pc-configuration) | PC Network Configuration | Configuring multiple PCs in a network |
| [Lab 03](./lab-03-eigrp-configuration) | EIGRP Routing Protocol | Implementing EIGRP for dynamic routing |
| [Lab 04](./lab-04-advanced-routing) | Advanced Routing & Subnetting | Complex subnetting scenarios and routing |
| [Lab 05](./lab-05-final-implementation) | Complete Network Implementation | Final comprehensive network design |

## 🛠️ Tools Used
- **Cisco Packet Tracer** - Network simulation tool
- Various network devices (Routers, Switches, PCs)
- Routing protocols (EIGRP, Static Routing)

## 📚 Key Concepts Covered
- IP addressing and subnetting
- VLSM (Variable Length Subnet Mask)
- Wildcard masks
- EIGRP configuration
- Static and dynamic routing
- Network troubleshooting with ping
- Switch and router configurations

## 🚀 How to Use
1. Install Cisco Packet Tracer (version 8.x or higher)
2. Clone this repository
3. Open any `.pkt` file with Packet Tracer
4. Explore the network topology and configurations
